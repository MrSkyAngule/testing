### Это сайт который работает в интернете по айпи uhrwerk.ru

## Чтобы скачать необходимые модули пайтена
``pip install -r requirements.txt``

## перед использованием надо настроить сайт

в settings:
``ALLOWED_HOSTS = ['ваши домены']``

и измените на свои данные в настройке базы данных


## чтобы запустить сайт надо
``make start``

## а чтобы остановить
``make stop``

### так как это рабочий сайт тут нету css и html
